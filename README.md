# mPushSDK 

mPushSDK for iOS


___
使用CocoaPods 添加

1.在你项目的 Podfile 中添加以下代码：

`pod ‘mPushLib’`

2.执行 pod 安装命令

`$ pod install` 或者 `$ pod update`
